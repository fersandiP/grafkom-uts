README Pang Pong
================

Program yang dibuat merupakan permainan sederhana mirip seperti pong.
Terdapat tiga komponen pada permainan ini : papan, bola, segitiga.
Pengguna mengendaliakan papan dan mengerakkannya ke atas/bawah sehingga bola tidak melewati papan tersebut.
Jika bola mengenai segitiga maka pengguna akan mendapatkan skor tambahan.
Permainan berakhir jika bola sampai pada dinding bagian kiri canvas.

Program sudah dapat dijalankan dengan membuka file index.html pada browser.
