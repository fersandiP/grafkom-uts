CLEAN+BUILD+RUN
==============================================================================
Telah tersedia Makefile untuk build di Linux/MacOS/Windows.
Program ini dibuat dan dicoba dengan MacOS dan belum dicoba di Linux/Windows.
Output dari build berada pada folder out/
Command:
-make : clean+build
-make clean : clean
-make run   : run
